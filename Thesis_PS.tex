    \begin{abstract}
   
There are vast amount of correlated neural activities, such as oscillations and synchronizations, observed in the brain. These activities are one of the most important keys for communication in the brain. However, the questions on how the synchronization level can be modulated and how these synchronized activities affect spike transfer from one layer to another layer in different convergent connection conditions are not clearly understood. In this work, we employ computer simulation of realistic neural network to address the questions. 

The neural network consists of two layers of conductance based single cell model, a source layer and a target layer. The interlayer connections follow statistical wiring rule for which, strength and connectivity of the connection depend on the distance between the target cell and the source cells which define as Gaussian-Gaussian (GG) convergent rule. For comparison, two other convergent rules that have constant probability of connection within the range were made. One rule has uniformly distribution of connection strength. This rule is defined as Uniform-Uniform (UU) convergent rule. Another rule has random connection strength that draw from negative exponential distribution and so it is named as Uniform-Exponential (UE) convergent rule. Then, the responses on target layer were measured from various convergent conditions. To study how the synchronization of input contribute to the neuron network's response, we compared the oscillating input and the static input given to the system.

We observed that, when oscillating input were given, output firing rate is higher than that of static input even though the overall firing rate of both input were same. Also, the specific level of response can be made with less strength of connection in oscillating input compare to the static input. We showed that the oscillating input induced more output spike than static input. In addition, the gain of response from static to oscillating input are largest in UU rule compare to GG and UE rules. We discovered that the output spikes in UU rule have the most dependency on synchronization level in input compare to GG and UE rules. These results showed that UU has the most sensitivity on the synchronization level in input.
 
In summary, we found that high level of synchronization in input resulted in high output response and the synchronized input can make a specific level of output with low cost of connection compare to the static input. Plus, neural network with Uniform-Uniform convergent connection rule is selective to change in input synchronization compare to the other two rules. These results suggest that both input synchronization level and the interlayer connection rules  can contribute to quest on understanding brain connection. 

    \end{abstract}

\begin{summary}

    지난 10여 년간 탄소 나노튜브는 자체의 독특한 전기적, 기계적 성질로
    인하여 다가오는 나노기술 분야의 이상적인 기초물질중의 하나로 떠오르고
    있다. 흑연을 감는 세세한 방법에 따라 전기적 특성이 금속성에서 1eV의
    띠간격을 가지는 반도체 특성까지 다양한 분포로 존재한다.
    본 학위논문에서는 탄소 나노튜브의 여러 물리적 성질에 대해 고찰하는데,
    기본적으로 제일원리 밀도함수 이론과 밀접결합근사 모형을 사용하여 전기적
    특성과 그 제어 방법, 자기적 특성, 그리고 수송특성 등을 다루고자 한다.

\end{summary}
